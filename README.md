# Formula 1 Historical Analysis (1950–2024)

## Project Overview

This project explores historical Formula 1 data using Python, Pandas, and Matplotlib.

The analysis answers the following questions:

1. Who are the top 10 drivers by race wins?
2. Which constructors have the most race wins?
3. Which countries have produced the most Formula 1 drivers?
4. How has the Formula 1 calendar grown over time?

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Key Findings

* Lewis Hamilton leads Formula 1 history with 105 race wins.
* Ferrari is the most successful constructor with 249 wins.
* British drivers represent the largest nationality group in Formula 1.
* Formula 1 has expanded significantly, with race counts increasing across decades.
